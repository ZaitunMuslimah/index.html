ZAITUN MUSLIMAH : Ringkasan materi 100 kata

HTML sebuah coding yang memerintah menggunakan kode-kode dan mengirim ke web agar menjadi sebuah tampilan sesuai dengan fungsinya, html merupakan kerangka atau struktur sebuah website. Dalam membuat sebuah HTML yang perlu diperhatikan adalah perintah-perintah yang ada didalamnya yaitu: ketika ingin mebuat nama file harus berindeks html, untuk coding di dalamnya juga ada type document yaitu html dan indeks html tetapi jika tidak ingin menggunakannya bisa saja untuk bagian code didalamnya. Css yaitu sebuah desain atau tampilan apda sebuah halaman website, misalnya seperti desain antarmuka pada web Politeknik Hasnur. Javascript yaitu interaksi antara kita dengan sebuah halaman ataupun halaman dan server, jadi Javascript membuat halaman web lebih dinamis dan interaktif, dalam urusan front end maupun back end diurus oleh javascript.
